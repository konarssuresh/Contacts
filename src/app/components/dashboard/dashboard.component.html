Contacts App:-

This is a sample app which is used to store contacts. This application makes use of localstorage used in the browser instead of making API calls which can be easily replaced in presence of APIs.

This App Performs following operations:-

  • View Contacts :- View the list of contacts stored.
  • Add New Contact :- Add a new contact to the List.
  • Edit Contact:- Edit the details of the contact.
  • Delete Contact:- To remove the contact from the list.
  • Activate/Inactivate Contact:- To change the status of contact.
